## 🎯 Overview

CrystieHunter is a powerful browser-based OSINT framework that combines real API integrations with a cyberpunk interface. Designed for security researchers and intelligence analysts.

> ⚠️ **Educational Purpose Only** - Use responsibly and only against targets you have permission to test.

---

## ✨ Key Features

### 🖧 Network Intelligence
| Module | Description |
|--------|-------------|
| DNS Lookup | A, MX, NS, TXT record enumeration |
| WHOIS Lookup | Domain registration details |
| GeoIP Tracking | Geographic location of IPs |
| Shodan Integration | Device intelligence & open ports |
| SSL/TLS Analysis | Certificate details |
| Subdomain Finder | Subdomain enumeration |
| Reverse DNS | PTR record lookup |

### 👥 Social Media Intelligence (150+ Platforms)
Facebook • Instagram • Twitter/X • LinkedIn • GitHub • TikTok • Snapchat • YouTube • Reddit • Telegram • Pinterest • Spotify • Twitch • Discord • Tumblr • Medium • DeviantArt • Behance • Dribbble • Flickr • VK • Weibo • SoundCloud • Steam • Xbox Live • PlayStation • Quora • Stack Overflow • HackerNews • Product Hunt • AngelList • Crunchbase • ResearchGate • Academia.edu • Keybase • About.me • Imgur • Giphy

### 📧 Email & Phone Intelligence
- Breach Detection via HaveIBeenPwned
- Email & Phone Validation
- Carrier & Location Analysis

### 🖼️ Metadata Extraction
- EXIF data extraction from images
- File information (size, type, dimensions)

### 🎯 Target Types Supported
🌐 Domain • 📍 IP Address • 👤 Username • 🧑 Person • 📧 Email • 📱 Phone • 🖼️ Image • 🏢 Company
